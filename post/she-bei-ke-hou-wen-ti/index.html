设备课后问题

2023-06-02

第一次：
1.组成网络的主要部件有：
终端设备，中间设备，网络介质

2.常见网络拓扑图有几种？各自特点？何为融合网络？
1）物理拓扑图：
物理拓扑图说明了中间设备和电缆安装的物理位置
2）逻辑拓扑图：
逻辑拓扑图说明了设备、端口和网络的编址方案
3）融合网络：
融合网络能够通过相同的网络基础设施，在许多不同类型的设备之间传输数据、语音和视频。此网络基础设施采用一组相同的规则、协议和实施标准。融合数据网络在一个网络中传送多种服务。

3.访问思科交换机的方法有那些？思科操作系统的命令模式有那些，要配置设备必须进入什么模式
控制台，ssh，telnet

用户 EXEC 模式 - 该模式功能有限，但可用于有效执行基本操作。它只允许有限数量的基本监控命令，不允许执行任何可能改变设备配置的命令。用户 EXEC 模式由采用 > 符号结尾的 CLI 提示符标识。
特权 EXEC 模式 - 要执行配置命令，网络管理员必须访问特权 EXEC 模式。较高级别的配置模式，比如全局配置模式，只能通过特权 EXEC 模式访问。特权 EXEC 模式由采用# 符号结尾的提示符标识。

要配置设备，用户必须进入全局配置模式。

4.配置特权模式密码的命令，加密密码的命令
enable secret password

service password-encryption

5.保存配置信息的命令是什么？为什么要用默认网关？
copy running-config startup-config
因为交换机是二层设备，没路由功能所需要指定路由功能路由器或者三层交换机址作默认网关

6.解释下面命令：

configure terminal                                      进入全局配置模式
interface vlan 1                                           进入 VLAN 1 的接口配置模式。
ip address   192.168.1.20 255.255.255.0     将 IPv4 地址配置为 192.168.1.20，将子网掩码配置为 255.255.255.0。         
no shutdown                                                启动接口
exit                                                               从全局配置模式的任何子配置模式切换到模式层级中的上一级模式，请输入exit命令。
ip default-gateway 192.168.1.1                   设置默认网关
