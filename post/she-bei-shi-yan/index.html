十、路由基本配置

10.1配置初始路由器设置

基本路由器配置步骤

1、配置设备名称

Switch(config)#hostname hostname

2、保护特权模式

Switch(config)#enable secret password

3、保护用户模式

Switch(config)#line console 0
Switch(config-line)#password password
Switch(config-line)#login

4、保护远程Telnet/SSH访问

Switch(config-line)#line vty 0 4       交换机（0  15）
Switch(config-line)#password password
Switch(config-line)#login
Switch(config-line)#transport input {ssh | telnet}

5、保护配置文件中的所有密码。

Switch(config-line)#exit
Switch(config)#service password-encryption

6、提供法律通知

Switch(config)#banner motd #delimiter message delimiter#

7、保存配置

Switch(config)#end
Switch#copy running-config startup-config

10.2配置接口

1、进入全局模式

R1#configure terminal

2、配置接口GigabitEthernet 0/0/0

R1(config)#interface gigabitethernet 0/0/0

交换机：
interface vlan 1
ip address ...

3、将链路描述为“Link to LAN”

R1(config-if)#description to Link to LAN

4、使用 IPv4 地址192.168.10.1和子网掩码255.255.255.0配置接口

R1(config-if)#ip address 192.168.10.1 255.255.255.0

5、使用 IPv6 地址 2001:db8:acad:10::1 和前缀长度**/64**配置接口

R1(config-if)#ipv6 address 2001:db8:acad:10::1/64

6、启用接口并返回全局配置模式。

R1(config-if)#no shutdown
R1(config-if)#exit

10.3配置默认网关

1、进入全局

S1#configure terminal

2、将 S1 的默认网关配置为“192.168.10.1”

S1(config)#ip default-gateway 192.168.10.1
